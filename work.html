学習スケジュール管理アプリ

Ebbing Learn

  • 概要

    効率的な学習をするためのスケジュール作成を補助するためのアプリケーションです。学習で重要なのは「反復と継続」ですが、これらを考慮して計画を立てようとすると、面倒でやらなかったり非効率な計画になったりとうまくいかないことが多いと思います。このアプリでは、そんな「反復・継続」の両方を実現する素晴らしい学習スケジュールを立てるのを手助けします。

  • 特徴

    勉強するモチベーションを保ち続けるために、目標を大・中・小に細分化して目の前の目標を分かりやすくする、終わったタスクが積み重なって目に見える形で成果を見れる、休憩日などを設けるなど様々な工夫をしています。

  • 開発環境

    プラットフォーム: Firebase DataBase: FireStore フレームワーク: React フロントエンド: HTML/Sass/JavaScript バックエンド: JavaScript

自動時間配分アプリ

Taskative

  • 概要

    日々変動する自由時間に対して、最適な各タスクの時間配分を提案します。

  • 特徴

    ユーザーがタスクに対して活動できる時間を入力すると、その時間に合わせて各タスクの時間配分が提案されます。タスクはユーザーによって追加・削除することができ、その他様々な設定を行うことで柔軟に時間配分を決めることができます。

  • 開発環境

    プラットフォーム: Firebase DataBase: FireStore フレームワーク: React フロントエンド: HTML/Sass/TypeScript バックエンド: TypeScript
